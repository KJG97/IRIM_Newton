 결론
──────────────────────────────────────────────────────────────────────────

 1. stiffness / damping / effort_limit: IsaacSim USD와 IsaacLab allex.py **완전 일치**
 2. armature: USD에 0이 아닌 값 설정 → IsaacLab에서는 미설정 (0으로 사용)
    → 동역학 응답(가속도, 관성 토크)에 차이를 줄 수 있음
 3. maxJointVelocity: 양쪽 일치 (팔 114.6 deg/s, 손 572.9 deg/s)
 4. 초기 관절 위치: 실질적으로 일치
 5. drive type: 양쪽 모두 "force" 모드

 ★ 토크 차이의 주 원인: PD 게인이 아닌, action 적용 타이밍/동기성 차이
   + armature 차이로 인한 동역학 응답 차이
